# AI-Enhanced To-Do App

A simple and interactive To-Do web application built using HTML, CSS, and JavaScript, featuring task management with persistent storage and AI-like task suggestions.

---

## Features

- Add new tasks  
- Mark tasks as completed  
- Edit existing tasks  
- Delete tasks  
- Save tasks using browser localStorage  
- Live task counter (Total / Completed)  
- AI-like smart task suggestions while typing  
- Deployed live using GitHub Pages  

---

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Browser localStorage  
- GitHub Pages

##  AI Logic Explanation

This project uses **rule-based AI logic** to provide intelligent task suggestions based on keywords typed by the user (e.g., study, health, project).  
The architecture allows easy future integration with real AI APIs such as OpenAI or Gemini.
